СПОСОБИ ПІДКЛЮЧЕННЯ СТИЛІВ

Існує три основні способи підключення CSS до HTML:

  1. Вбудований стиль (inline CSS) – додається безпосередньо в атрибут style елемента.
    <p style="color: red;">Цей текст червоний</p>

    Цей текст червоний
  2. Внутрішній стиль (internal CSS) – прописується у <style> всередині <head>
    <style>
    p { color: blue; }
    </style>
  3. Зовнішній стиль (external CSS) – підключається через окремий файл .css за допомогою
    <link rel="stylesheet" href="styles.css">

СЕЛЕКТОРИ CSS

Селектор – визначає до якого HTML-елемента буде застосований стиль CSS. Селекторами можуть бути HTML-елементи, їх класи та ідентифікатори, а також псевдокласи та псевдоелементи.

СЕЛЕКТОР ТЕГУ

Селектори тегу (Type Selector) – застосовують стилі до всіх елементів певного типу.

p { color: green; }

СЕЛЕКТОР КЛАСУ

Селектори класу (Class Selector) – використовують . перед назвою класу для вибору всіх елементів із цим класом.

.highlight { background-color: yellow; }

<p class="highlight">Цей текст має жовте тло.</p>

Цей текст має жовте тло.

СЕЛЕКТОР ІДЕНТИФІКАТОРА

Селектори ідентифікаторів (ID Selector) – використовують # перед ідентифікатором, застосовується до одного унікального елемента.

#header1 { font-size: 24px; }

<h1 id="header1">Заголовок</h1>

ВИСНОВОК

У процесі вивчення теми "Каскадні таблиці стилів. Селектори. Ідентифікатори. Стильове оформлення текстових елементів в HTML-документах" було розглянуто основи використання CSS для створення сучасного і структурованого веб-дизайну. Завдяки практичним завданням набуті навички роботи з селекторами, включаючи селектори тегів, класів та ідентифікаторів, а також комбіновані селектори, які допомагають точно визначати, до яких елементів застосовуються стилі. Освоєно використання кольорів, фонів, відступів (margin, padding), плаваючих елементів (float), що дозволяє створювати адаптивні та привабливі макети веб-сторінок. Окрему увагу приділено стилізації текстових елементів, зокрема змінам шрифтів, розміру тексту, міжрядкового інтервалу та оформленню заголовків, абзаців і списків. Отримані знання та практичний досвід дозволяють ефективно застосовувати CSS у розробці веб-інтерфейсів, роблячи їх зручними та естетично привабливими.
